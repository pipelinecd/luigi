luigi
=====

Developer web-UI

To get started:

    sudo npm install -g yo grunt-cli bower karma
    npm install  
    bower install  

To run the application
    
    grunt serve [--force]

To run tests
    
    cd ./test
    karma start

Errors
------
If you get 
   
    TypeError: Cannot read property 'stdout' of undefined

Run:
   
    gem install compass

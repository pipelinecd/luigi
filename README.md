luigi
=====

Developer web-UI

To get started:

    sudo npm install -g yo grunt-cli bower karma
    npm install  
    bower install  

To run the application:

    grunt serve [--force]

To run tests, from the root:

    karma start

And to run the tests:

    karma run


Errors
------
If you get 
   
    TypeError: Cannot read property 'stdout' of undefined

Run:
   
    gem install compass
